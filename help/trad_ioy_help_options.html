Gotta love options, eh? Traditores In-One-Year is a simple Wordpress plugin, but it has a decent amount of options (described below)...

Page/Post Setting

There is a single setting avaialble for each post and page. It's purpose is to keep Traditores In-One-Year's impact minimal by only loading CSS and JavaScript libraries (and doing shortcode substitutions) when you really want it to. It shows up under "Traditores IOY."

Activate Shortcodes

Check this if you want to use the Traditores In-One-Year shortcode.

Primary Settings

Bible Version

Choose the default Bible version.

Page Behavior

Three checkboxes:

Scroll to today's entry?

Check if you want the page to automatically scroll down to today's Bible reading.

Copy current day's row to top of table?

Check if you want today's Bible reading row in the table to be copied above the top row of the table.

Keep user's Bible preference in a cookie?

Check if you want to keep your visitor's preferred Bible version in a cookie (so they don't have to choose it every time they return).

Custom CSS Settings

Header <Style>

Put CSS code here you want inserted into the head section of the web page. Please pay special attention to the warning just beneath the setting.

CSS File

The path to a CSS file you want loaded. Please pay special attention to the information just beneath the setting.

JavaScript and CSS Settings

Five checkboxes:

Use minimized css and js files where possible?

Check to use the smaller, minimized versions of CSS and JavaScript files. (Recommended unless you are debugging an issue.)

Load Traditores In-One-Year css?

Check to use the CSS formatting provided in this plugin. (Recommended unless you want to totally override the look in feel. See the "CSS Classes" section of the "Instructions" help tab.)

Load jQuery cookie extension?

Check to use load the JavaScript for the jQuery cookie extension. (Only uncheck if you already are loading it or you will never set Traditores In-One-Year to save the user's Bible preference in a cookie.)

Load jQuery scrollTo extension?

Check to use load the JavaScript for the scrollTo extension. (Only uncheck if you already are loading it or you will never set Traditores In-One-Year to scroll to today's reading.)

"Be Especially Cautious" Settings

Delete Settings

Only check this if you want to permanently uninstall Traditores In-One-Year.
